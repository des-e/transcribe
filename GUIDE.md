# Транскрибация — Инструкция по установке

Инструмент для перевода видео и аудио в текст. Работает локально, без интернета (после установки).

---

## Mac (M1 / M2 / M3)

### Установка (один раз)

**Шаг 1 — Клонировать репозиторий**

```bash
git clone https://github.com/des-e/transcribe.git ~/transcribe
```

> Если `git` не установлен — macOS предложит установить Xcode Command Line Tools. Согласиться и повторить команду.

**Шаг 2 — Создать иконку для Dock**

```bash
bash ~/transcribe/make_app.sh
```

В папке `~/transcribe` появится `Transcribe.app`.

**Шаг 3 — Добавить в Dock**

Открыть Finder → папка `transcribe` → перетащить `Transcribe.app` в Dock.

---

### Первый запуск

1. Кликнуть на иконку в Dock
2. macOS спросит «Открыть?» → нажать **Открыть** (один раз)
3. Откроется Terminal и начнётся загрузка модели Whisper (~800 MB)
4. Прогресс виден в терминале — дождаться завершения
5. Автоматически откроется браузер с приложением

> Загрузка модели происходит **один раз**. Все следующие запуски мгновенные.

---

### Ежедневное использование

Кликнуть на `Transcribe.app` в Dock → браузер откроется автоматически.

**Или из терминала:**
```bash
bash ~/transcribe/start.sh
```

---

### Обновления

Происходят автоматически при каждом запуске. Ничего делать не нужно.

---

### Устранение проблем

**Приложение не открывается (Gatekeeper)**
```bash
xattr -cr ~/transcribe/Transcribe.app
```

**Порт занят (сервер не запускается)**
```bash
kill -9 $(lsof -t -i :8000) 2>/dev/null
```

**Пересоздать Transcribe.app**
```bash
bash ~/transcribe/make_app.sh
```

**Ошибка при git clone: `xcrun ... incompatible architecture`**

Возникает на M-чипах с устаревшими Xcode Command Line Tools (чаще всего после переноса с Intel Mac).

```bash
sudo rm -rf /Library/Developer/CommandLineTools
xcode-select --install
```

После установки повторить `git clone`.

## Windows

### Установка (один раз)

**Вариант А — через Git**

```
git clone https://github.com/des-e/transcribe.git C:\transcribe
```

**Вариант Б — скачать архив**

1. Открыть `https://github.com/des-e/transcribe`
2. Нажать **Code → Download ZIP**
3. Распаковать в `C:\transcribe`

---

### Первый запуск

1. Открыть папку `C:\transcribe`
2. Дважды кликнуть `start.bat`
3. Откроется консоль — начнётся установка `uv` и загрузка модели Whisper (~1.5 GB)
4. Прогресс виден в консоли — дождаться завершения
5. Автоматически откроется браузер с приложением

> Загрузка модели происходит **один раз**. Все следующие запуски мгновенные.

---

### Ежедневное использование

Дважды кликнуть `start.bat` в папке проекта.

**Ярлык на рабочем столе:**
Правой кнопкой на `start.bat` → **Отправить → Рабочий стол (создать ярлык)**.

---

### Обновления

Происходят автоматически при каждом запуске. Ничего делать не нужно.

---

### Устранение проблем

**Консоль мгновенно закрывается**
Запустить через командную строку:
```
cd C:\transcribe
start.bat
```

**Порт занят**
```
netstat -ano | findstr :8000
taskkill /PID <номер> /F
```

## Поддерживаемые форматы

| Тип | Форматы |
|-----|---------|
| Видео | MP4, MKV, AVI, MOV, WEBM |
| Аудио | MP3, M4A, WAV, OGG, AAC, FLAC |

Максимальный размер файла: **2 GB**

---

## Модели Whisper

| Модель | Скорость | Качество | Рекомендуется |
|--------|----------|----------|---------------|
| Tiny | очень быстро | базовое | для тестов |
| Base | быстро | стандартное | |
| Small | средне | хорошее | |
| **Medium** | медленнее | **отличное** | ✓ по умолчанию |
| Large | долго | максимальное | для важных записей |

На **Mac M1/M2/M3** используется MLX-бэкенд — в 4-8× быстрее CPU.
