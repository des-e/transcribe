# Транскрибация

Локальный инструмент транскрибации видео и аудио. Веб-интерфейс, работает без интернета после первой загрузки модели.

**Движки:**
- **Apple Silicon (M1/M2/M3)** — mlx-whisper (быстро, нативно)
- **Windows / Linux / Intel Mac** — faster-whisper (CPU, int8)

## Установка

### Mac

**Шаг 1.** Открой Терминал (`Cmd+Пробел` → "Терминал")

**Шаг 2.** Клонируй репозиторий:
```bash
git clone https://github.com/des-e/transcribe.git ~/transcribe
```

> Если git не установлен — появится окно с предложением установить Xcode Command Line Tools. Нажми «Установить» и дождись окончания.

**Шаг 3.** Запусти:
```bash
bash ~/transcribe/start.sh
```

При первом запуске автоматически установятся:
- **uv** — менеджер Python-окружений
- **Homebrew** — если не установлен (потребуется пароль администратора)
- **ffmpeg** — для видеофайлов
- **Модель Whisper medium** (~1.5 GB, скачивается один раз)

**Шаг 4.** Открой браузер: [http://localhost:8000](http://localhost:8000)

---

**Опционально: иконка в Dock**

Чтобы запускать одним кликом из Dock:
```bash
bash ~/transcribe/make_app.sh
```
Создаст `Transcribe.app` на Рабочем столе — перетащи его в Dock.

---

### Windows

**Шаг 1.** Установи [Git для Windows](https://git-scm.com/download/win), если не установлен

**Шаг 2.** Открой PowerShell и клонируй репозиторий:
```powershell
git clone https://github.com/des-e/transcribe.git $env:USERPROFILE\transcribe
```

**Шаг 3.** Запусти `start.bat` двойным кликом

При первом запуске автоматически установится uv и скачается модель.

> ffmpeg нужен для видеофайлов. Установи отдельно:
> ```
> winget install Gyan.FFmpeg
> ```
> После установки перезапусти `start.bat`.

**Шаг 4.** Открой браузер: [http://localhost:8000](http://localhost:8000)

## Обновление

Обновления применяются **автоматически** при каждом запуске — скрипт проверяет GitHub и обновляет файлы.

Если нужно обновить вручную:
```bash
cd ~/transcribe && git pull
```

## Использование

1. Перетащи файл или нажми на зону загрузки
2. Выбери модель и язык (RU / EN / Авто)
3. Нажми **Транскрибировать**
4. Кликай по временным меткам — аудиоплеер перемотает к нужному месту
5. Текст сегментов можно редактировать прямо в интерфейсе (клик по тексту)
6. Экспортируй результат в **TXT** или **SRT**

## Модели

| Модель | Качество | Скорость |
|--------|----------|----------|
| `tiny`   | низкое   | очень быстро |
| `base`   | приемлемо | быстро |
| `small`  | хорошее  | сбалансировано |
| `medium` | отличное | рекомендуется ← |
| `large`  | максимум | медленно |

Первый запуск выбранной модели скачает и закеширует её. Последующие запуски — мгновенные.

## Форматы файлов

**Видео** (требует ffmpeg): MP4, MKV, AVI, MOV, WEBM  
**Аудио**: MP3, M4A, WAV, OGG, AAC, FLAC
