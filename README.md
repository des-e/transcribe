# Транскрибация

Локальный инструмент транскрибации видео и аудио. Веб-интерфейс, работает без интернета после первой загрузки модели.

**Движки:**
- **Apple Silicon (M1/M2/M3)** — mlx-whisper (быстро, нативно)
- **Windows / Linux / Intel Mac** — faster-whisper (CPU, int8)

## Установка

### Mac

1. Скачай папку `transcribe` в домашний каталог (`~/transcribe`)
2. Запусти двойным кликом `start.command`

При первом запуске автоматически установятся:
- [uv](https://github.com/astral-sh/uv) — менеджер Python-окружений
- [Homebrew](https://brew.sh) — если не установлен
- [ffmpeg](https://ffmpeg.org) — для видеофайлов
- Модель Whisper `medium` (~1.5 GB, один раз)

### Windows

1. Скачай папку `transcribe`
2. Запусти `start.bat`

При первом запуске установятся uv и модель. ffmpeg нужен отдельно:
```
winget install Gyan.FFmpeg
```

## Запуск

| Платформа | Команда |
|-----------|---------|
| Mac (двойной клик) | `start.command` |
| Mac (терминал) | `bash start.sh` |
| Windows | `start.bat` |

Открой браузер на [http://localhost:8000](http://localhost:8000)

## Использование

1. Перетащи файл или нажми на зону загрузки
2. Выбери модель и язык
3. Нажми **Транскрибировать**
4. Кликай по временным меткам — аудиоплеер перемотает к нужному месту
5. Текст сегментов можно редактировать прямо в интерфейсе
6. Экспортируй в **TXT** или **SRT**

## Модели

| Модель | Качество | Скорость |
|--------|----------|----------|
| `tiny`   | низкое   | очень быстро |
| `base`   | приемлемо | быстро |
| `small`  | хорошее  | сбалансировано |
| `medium` | отличное | рекомендуется ← |
| `large`  | максимум | медленно |

## Форматы файлов

**Видео** (требует ffmpeg): MP4, MKV, AVI, MOV, WEBM  
**Аудио**: MP3, M4A, WAV, OGG, AAC, FLAC
