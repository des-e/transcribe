# Транскрибация созвонов

Локальная транскрибация видео и аудио на русском языке.
Веб-интерфейс на Gradio + faster-whisper. Бесплатно, без интернета после первой загрузки модели.

## Требования

- Python 3.11+
- ffmpeg: `winget install Gyan.FFmpeg`

## Установка (один раз)

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Автоматически откроется браузер на `http://localhost:7860`.

## Использование

1. Перетащи видео или аудио файл в поле загрузки
2. Выбери модель (рекомендуется `medium`)
3. Нажми **Транскрибировать**
4. После — **Сохранить рядом с файлом** (создаст `.txt` рядом с видео)

## Модели

| Модель | ~Время на 1 час | Качество |
|--------|----------------|----------|
| `tiny`   | 3 мин  | низкое   |
| `base`   | 5 мин  | приемлемо|
| `small`  | 10 мин | хорошее  |
| `medium` | 20 мин | отличное ← по умолчанию |
| `large`  | 40 мин | максимум |

Первый запуск выбранной модели скачает её (~1.5 GB для medium) и закеширует локально.

## Командная строка (без интерфейса)

```bash
python transcribe.py meeting.mp4 --extract-audio
```

## Форматы файлов

mp4, mkv, avi, mov, mp3, m4a, wav, ogg
